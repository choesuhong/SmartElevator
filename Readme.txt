얼굴인식을 통한 인공지능 엘리베이터
버튼을 눌러 작동시키는 기존의 엘리베이터에 얼굴인식 기능을 추가하여 자동으로 원하는 층 수를 눌러주는 인공지능 엘리베이터 ○ 엘리베이터 작동이 어려운 사람에게 미리 등록된 승객정보를 이용해 얼굴인식을 통한 승객들의 층수를 자동으로 눌러주는 편의 기능 제공 ○ 음성인식 기능과 SMS 기능을 추가하여 기존의 엘리베이터보다 더욱 편리하게 이용가능하며 불편점 해소 ○ 관리자 측면에서도 시스템 서버 내에 이용자의 정보 및 승객의 사진이 포함되어있어 웹 서비스를 통한 이용자 관리에 수월



○ Java_Main_Server : 얼굴인식 메인 로직서버와 아두이노, 안드로이드 연동코드

○ Android : 엘리베이터 인터페이스 구현

○ Web_Server : 사용자 관리 및 얼굴인식 엘리베이터 웹서비스 구현

○ Swing_Server : [구 버전] 웹 서비스 구현 전 Swing을 이용해 사용자 관리
